Integration Protocols

x402 API

HTTP-based protocol. Call an endpoint, get a 402 with pricing, then pay with a signed payment header.

GET /api/v1/agent/hire?description=...&winners=10&perUser=0.025
Get Key

MCP Server

Model Context Protocol — works with Claude Desktop, Cursor, and any MCP client.

endpoint: https://ogapay-five.vercel.app/api/mcp

OpenClaw Skill

Install the OgaPay skill into your OpenClaw-compatible agent for seamless integration.

curl -s https://ogapay.fun/skill.md
